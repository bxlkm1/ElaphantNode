Elephant Node
===================================

.. image:: logo.png
    :width: 120px
    :alt: elastos logo
    :align: center

The Elephant node is mainly used for the elephant wallet, the Elephant node itself contains the main chain node of the ELA and the ID chain node. Elephant Node extends features which Elephant wallet requires.

.. warning::
    Since software is written by humans, it can have bugs. Thus, if you have questions you are always welcome to issue your problem.
